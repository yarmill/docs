Files collects every attachment from the Plan and the Training Log — videos, images,
documents — in one overview, so "that technique video from camp" is findable months
later.

**For:** coaches, athletes · **Where:** Web app

## Find a file

Open **Files** for the overview: every file with its **type**, **name**, **tags**,
**location** (where in the log it lives), and **date** — each column filterable. Select
a file's date to jump to its day in the log.

There are deliberately **no folders**. Folders force one hierarchy; **tags** let the
same race video be "technique" and "prone shooting" at once. Tag consistently and
filtering does the rest.

## Add a file

Attach files directly on a day in the [Training Log](/en/reality/training-log) or Plan —
or select **Add file** here, which also lets you attach one file (a shared video, say)
**to several athletes at once**, with its date and tags.
